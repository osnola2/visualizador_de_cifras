# Mapeamento Completo do Acorde de Dó Maior (C) no Cavaquinho 🪕

Este guia apresenta todas as possíveis digitações, inversões e formas do acorde de **Dó Maior (C)** ao longo do braço do cavaquinho na afinação tradicional brasileira: **Ré (4ª) - Sol (3ª) - Si (2ª) - Ré (1ª)** (`D4 - G4 - B4 - D5`).

---

## 1. A Tríade de Dó Maior (Teoria)

O acorde de Dó Maior é uma **tríade maior** formada por três notas:
- **Tônica (T)**: `Dó` (C)
- **3ª Maior (3M)**: `Mi` (E)
- **5ª Justa (5J)**: `Sol` (G)

Como o cavaquinho possui **4 cordas**, uma das notas da tríade sempre precisará ser repetida (dobrada) em qualquer digitação completa de 4 cordas.

### Mapa das Notas da Tríade no Braço do Cavaquinho

| Casa | 4ª Corda (Ré/D) | 3ª Corda (Sol/G) | 2ª Corda (Si/B) | 1ª Corda (Ré/D) |
|:---:|:---:|:---:|:---:|:---:|
| **0 (Solta)** | Ré | **Sol (5J)** | Si | Ré |
| **1ª** | D# / Eb | G# / Ab | **Dó (T)** | D# / Eb |
| **2ª** | **Mi (3M)** | Lá | C# / Db | **Mi (3M)** |
| **3ª** | Fá | A# / Bb | Ré | Fá |
| **4ª** | F# / Gb | Si | D# / Eb | F# / Gb |
| **5ª** | **Sol (5J)** | **Dó (T)** | **Mi (3M)** | **Sol (5J)** |
| **6ª a 7ª** | G# / Lá | C# / Ré | Fá / F# | G# / Lá |
| **8ª** | A# / Bb | D# / Eb | **Sol (5J)** | A# / Bb |
| **9ª** | Si | **Mi (3M)** | G# / Ab | Si |
| **10ª** | **Dó (T)** | Fá | Lá | **Dó (T)** |
| **12ª** | Ré | **Sol (5J)** | Si | Ré |
| **13ª** | D# / Eb | G# / Ab | **Dó (T)** | D# / Eb |
| **14ª** | **Mi (3M)** | Lá | C# / Db | **Mi (3M)** |

## 2. As 4 Formas Principais de C ao Longo do Braço (Sistema de Digitação)

Ao explorar todo o braço do cavaquinho (da 1ª à 14ª casa), encontramos **4 formas primárias** para tocar o acorde de Dó Maior em 4 cordas. Essas posições derivam da transposição de formatos abertos clássicos do cavaquinho:

### 🎸 Posição 1: Forma Aberta (1ª Casa) — 1ª Inversão (Baixo em Mi)
Esta é a posição mais comum e tradicional do cavaquinho, ensinada em todos os métodos de iniciação e usada no samba, choro e pagode.

* **Digitação (Trastes)**: `[2, 0, 1, 2]`
* **Notas (da 4ª para 1ª corda)**: `Mi (E) - Sol (G) - Dó (C) - Mi (E)`
* **Inversão**: **1ª Inversão** (pois a nota mais grave, na 4ª corda, é a 3ª do acorde: `Mi`).
* **Dedilhação Recomendada**:
  - 4ª corda (casa 2): Dedo Médio (2)
  - 3ª corda (casa 0): Corda Solta (O)
  - 2ª corda (casa 1): Dedo Indicador (1)
  - 1ª corda (casa 2): Dedo Anelar (3)

```
Posição 1: C [2, 0, 1, 2]
    D   G   B   D
  |---|---|---|---|
1 |   |   | (1)|   |  <- Dó (C) na 2ª corda
  |---|---|---|---|
2 | (2)| O |   | (3)|  <- Mi (E) na 4ª e 1ª corda
  |---|---|---|---|
3 |   |   |   |   |
  |---|---|---|---|
```

---

### 🎸 Posição 2: Forma de Lá (5ª Casa) — 2ª Inversão (Baixo em Sol)
Derivada do formato de **Lá Maior (A)** `[2, 2, 2, 2]`, transposta 3 casas acima. É extremamente confortável para repiques, palhetadas rápidas e convenções rítmicas do partido-alto.

* **Digitação (Trastes)**: `[5, 5, 5, 5]`
* **Notas (da 4ª para 1ª corda)**: `Sol (G) - Dó (C) - Mi (E) - Sol (G)`
* **Inversão**: **2ª Inversão** (pois a nota mais grave, na 4ª corda, é a 5ª do acorde: `Sol`).
* **Dedilhação Recomendada**:
  - Pestana reta na 5ª casa cobrindo todas as 4 cordas com o Dedo Indicador (1) ou Dedo Médio (2).

```
Posição 2: C [5, 5, 5, 5] (Pestana na 5ª casa)
    D   G   B   D
  |---|---|---|---|
4 |   |   |   |   |
  |---|---|---|---|
5 |(1)|(1)|(1)|(1)|  <- Pestana na 5ª casa (G - C - E - G)
  |---|---|---|---|
6 |   |   |   |   |
  |---|---|---|---|
```

---

### 🎸 Posição 3: Forma de Fá (8ª a 10ª Casa) — Posição Fundamental (Baixo em Dó)
Derivada do formato de **Fá Maior (F)** `[3, 2, 1, 3]`, transposta 7 casas acima. É a **única posição em 4 cordas no cavaquinho** onde a **Tônica (Dó/C)** está na corda mais grave!

* **Digitação (Trastes)**: `[10, 9, 8, 10]`
* **Notas (da 4ª para 1ª corda)**: `Dó (C) - Mi (E) - Sol (G) - Dó (C)`
* **Inversão**: **Posição Fundamental** (Baixo em Dó na 10ª casa da 4ª corda).
* **Dedilhação Recomendada**:
  - 4ª corda (casa 10): Dedo Mínimo (4) ou Anelar (3)
  - 3ª corda (casa 9): Dedo Médio (2)
  - 2ª corda (casa 8): Dedo Indicador (1)
  - 1ª corda (casa 10): Dedo Mínimo (4)

```
Posição 3: C [10, 9, 8, 10]
    D   G   B   D
  |---|---|---|---|
8 |   |   | (1)|   |  <- Sol (G) na 2ª corda
  |---|---|---|---|
9 |   | (2)|   |   |  <- Mi (E) na 3ª corda
  |---|---|---|---|
10| (4)|   |   | (4)|  <- Dó (C) na 4ª e 1ª corda
  |---|---|---|---|
```

---

### 🎸 Posição 4: Oitava da Posição Aberta (12ª a 14ª Casa) — 1ª Inversão Aguda
É a repetição exata da Posição 1, mas tocada uma oitava acima (12 casas adiante). Muito utilizada em arranjos virtuosos e soladas de choro (estilo Waldir Azevedo / Luciana Rabello).

* **Digitação (Trastes)**: `[14, 12, 13, 14]`
* **Notas (da 4ª para 1ª corda)**: `Mi (E) - Sol (G) - Dó (C) - Mi (E)`
* **Inversão**: **1ª Inversão Aguda**.
* **Dedilhação Recomendada**:
  - 3ª corda (casa 12): Dedo Indicador (1) no Sol
  - 2ª corda (casa 13): Dedo Médio (2) no Dó
  - 4ª e 1ª cordas (casa 14): Dedos Anelar (3) e Mínimo (4) nos Mi.

```
Posição 4: C [14, 12, 13, 14]
    D   G   B   D
  |---|---|---|---|
12|   | (1)|   |   |  <- Sol (G) na 12ª casa
  |---|---|---|---|
13|   |   | (2)|   |  <- Dó (C) na 13ª casa
  |---|---|---|---|
14| (3)|   |   | (4)|  <- Mi (E) na 14ª casa
  |---|---|---|---|
```

## 3. Tríades e Voicings de 3 Cordas (Sem a 4ª Corda / Para Solos)

Muitas vezes, em soladas, contrapontos ou quando o violão 7 cordas já está fazendo os baixos, o cavaquinista toca **apenas as 3 cordas mais agudas (`Sol - Si - Ré`)**. Isso produz aberturas extremamente limpas:

| Traste (`[X, 3ª, 2ª, 1ª]`) | Notas (`Sol - Si - Ré`) | Inversão / Função |
|:---:|:---:|:---|
| **`[X, 0, 1, 2]`** | `Sol (G) - Dó (C) - Mi (E)` | Tríade na 1ª casa (2ª Inversão aguda) |
| **`[X, 5, 5, 5]`** | `Dó (C) - Mi (E) - Sol (G)` | Tríade na 5ª casa (Fundamental pura nas 3 cordas agudas) |
| **`[X, 9, 8, 10]`** | `Mi (E) - Sol (G) - Dó (C)` | Tríade na 8ª casa (1ª Inversão aguda nas 3 cordas) |
| **`[X, 12, 13, 14]`** | `Sol (G) - Dó (C) - Mi (E)` | Tríade na 12ª casa (Repetição oitavada da 1ª casa) |

## 4. Família e Acordes Derivados de Dó (C) no Cavaquinho

A partir da forma básica de Dó na 1ª casa ou na 5ª casa, pequenos movimentos de um ou dois dedos geram toda a família de acordes harmônicos de Dó:

| Acorde | Digitação (Trastes) | Notas | Como Deriva de C |
|:---|:---:|:---:|:---|
| **C** (Dó Maior) | `[2, 0, 1, 2]` | `E - G - C - E` | Acorde base |
| **C6** (Dó com 6ª) | `[2, 0, 1, 0]` ou `[X, 2, 1, 2]` | `E - G - C - D` | Solta-se a 1ª corda (Ré/D é a 6ª de Dó) |
| **C7M** (Dó com 7ª Maior) | `[5, 4, 5, 5]` (3ª casa) | `G - B - E - G` | Forma mais popular e harmônica (sem tônica) |
| **C7** (Dó com 7ª Menor) | `[2, 3, 1, 2]` | `E - Bb - C - E` | Dedo 3 ou 4 vai na casa 3 da 3ª corda (Bb) |
| **C9 / Cadd9** | `[0, 0, 1, 2]` | `D - G - C - E` | Solta-se a 4ª corda (Ré/D no baixo como 9ª) |
| **C4 / Csus4** | `[3, 0, 1, 3]` | `F - G - C - F` | Dedo na 3ª casa da 4ª e 1ª corda (Fá é a 4ª de Dó) |
| **C/G** (Baixo em Sol) | `[5, 5, 5, 5]` ou `[5, 0, 1, 2]` | `G - C - E - G` | Pestana na 5ª casa ou baixo na 5ª casa da 4ª corda |

## 5. Dicas de Aplicação Prática no Samba e Choro

1. **Variação Dinâmica na Batida**: Em um samba longo em Dó maior, em vez de ficar apenas batendo na **Posição 1 `[2, 0, 1, 2]`**, alterne periodicamente para a **Posição 2 `[5, 5, 5, 5]`** no contratempo para criar brilho, ginga e variação tímbrica.
2. **Subida de Convenção**: Quando a música tem uma subida cromática ou chamada dramática (por exemplo, subindo de `C` para `D` e depois `E`), usar **`[5, 5, 5, 5]` (C)** -> **`[7, 7, 7, 7]` (D)** -> **`[9, 9, 9, 9]` (E)** gera um impacto sonoro espetacular.
3. **Ponte Melódica no Choro**: Em choros e valsas, conectar `[2, 0, 1, 2]` (1ª casa) com `[10, 9, 8, 10]` (10ª casa) passando pela pestana da 5ª casa `[5, 5, 5, 5]` cobre toda a extensão vocal e harmônica do cavaquinho com maestria.
